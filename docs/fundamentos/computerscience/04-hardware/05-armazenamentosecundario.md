# Armazenamento Secundário

Memória principal e CPU são voláteis, ou seja, perdem tudo quando são desligadas. Sem um armazenamento secundário, o computador perderia todos os dados, inclusive o sistema operacional e os aplicativos, a cada desligamento. Para resolver isso existe o **armazenamento secundário**, que é não volátil e retém dados mesmo sem energia.

Diferente da RAM, o armazenamento secundário não é diretamente endereçável pela CPU. É muito mais barato por byte que a RAM, permitindo grande capacidade, mas também muito mais lento.

Os dois tipos mais comuns atualmente são o **HDD** (hard disk drive), que armazena dados magneticamente em um disco giratório, e o **SSD** (solid-state drive), que armazena dados por cargas elétricas em células de memória não voláteis. SSDs são mais rápidos, silenciosos e resistentes a falhas mecânicas por não terem partes móveis.

Quando o computador é ligado, o sistema operacional é carregado do armazenamento secundário para a memória principal. O mesmo acontece com aplicativos e dados do usuário, tudo precisa ser carregado na RAM antes de ser usado.

:::tip
Geralmente, ao ler textos em inglês, você perceberá o uso da palavra memória (*memory*) para se referir à memória principal, volátil. Já o armazenamento (*storage*) costuma ser usado para se referir ao armazenamento secundário, persistente.
:::
