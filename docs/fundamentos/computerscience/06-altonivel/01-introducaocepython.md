# Introdução ao C e Python

Neste capítulo vamos introduzir duas linguagens de programação de alto nível, C e Python. Essa combinação serve para demonstrar como linguagens de programação tendem a ter funcionalidades similares, porém de formas diferentes.

A linguagem de programação C tem origem no início da década de 1970, quando foi utilizada para escrever uma versão do sistema operacional Unix. Apesar de ser conceitualmente uma linguagem de alto nível, C não está tão distante do código de máquina subjacente, o que a torna uma boa escolha para desenvolvimento de sistemas operacionais e outros softwares que fazem interface direta com o hardware. Uma atualização do C, conhecida como C++, veio à tona em meados da década de 1980. C e C++ são linguagens poderosas que podem ser utilizadas para cumprir quase qualquer tarefa em um computador. De qualquer forma, ainda são linguagens complexas que não proporcionam muitos recursos contra erros do programador, e mesmo assim seguem sendo boas escolhas para programas que precisam se manter próximos ao hardware, seja para obter performance, seja para gerenciar componentes físicos. Além do mais, programar em C é educacionalmente eficiente pois fornece um mapeamento direto entre conceitos de baixo e alto nível, e essa é a principal razão pela qual currículos de graduação e livros técnicos de computação optam por utilizá-lo ao abordar o tema da programação.

Classificar C como linguagem de alto nível geralmente gera confusão, pois ele é muito mais próximo do hardware do que outras linguagens como Python. Lançado na década de 1990, Python tem se tornado cada vez mais popular por ser simples de ler e escrever, ao passo que provê tudo que é necessário para criar softwares complexos. Escrever programas em Python é extremamente simples, mesmo sem experiência de programação é possível entender o que um trecho de programa faz apenas por saber o básico de inglês. Python acabou se tornando uma escolha natural para iniciantes em programação, e também para quem foi traumatizado pela complexidade de C, não será o seu caso, mantenha-se firme comigo.

Agora vamos passear por alguns dos elementos mais comuns entre a maioria das linguagens de alto nível. Vale destacar que o objetivo não é te tornar um programador apenas com esse capítulo, a intenção é que você se familiarize com as ideias mais comuns que podem ser encontradas em linguagens de programação no geral.

Lembre-se de que tudo que uma linguagem de programação de alto nível nos proporciona é uma abstração das instruções de CPU, e partindo desse ponto, vamos nos aprofundar em como essas linguagens tornam mais acessível o que está acontecendo por baixo dos panos no hardware.
