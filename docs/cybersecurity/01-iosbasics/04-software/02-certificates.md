# Developer Certificates

Os Developer Certificates são um mecanismo oficial da Apple usado para garantir que aplicações executadas no iOS tenham uma origem confiável. Todo app precisa ser assinado digitalmente com um certificado emitido pela Apple, o que vincula aquele binário a um desenvolvedor identificado.

Não é possível instalar ou executar aplicações arbitrárias no dispositivo sem uma assinatura válida, e qualquer modificação feita no binário depois de assinado invalida essa assinatura. Esse modelo garante rastreabilidade do desenvolvedor responsável por cada app e impede a execução de código não autorizado no sistema.
