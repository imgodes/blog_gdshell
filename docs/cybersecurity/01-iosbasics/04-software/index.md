# Segurança de Software

Depois da segurança de hardware, o próximo nível a entender é como o iOS protege o software que roda em cima dele. Enquanto o Secure Enclave e o APFS cuidam de chaves e dados em repouso, essa camada trata de isolar aplicações entre si, garantir que só código confiável seja executado e manter essas garantias válidas mesmo depois que o sistema já está rodando.

Os próximos tópicos cobrem o *sandboxing* de apps, o modelo de certificados de desenvolvedor e os mecanismos que mantêm o kernel e o runtime íntegros ao longo do tempo, incluindo o ciclo de atualizações que sustenta tudo isso.
