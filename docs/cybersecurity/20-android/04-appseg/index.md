---
title: Apps e Segurança
---

Um app Android é distribuído como um APK, um pacote compilado a partir de código Java ou Kotlin, assinado digitalmente e executado sobre a Dalvik VM ou o ART. Esta seção cobre a estrutura interna do APK, o processo de compilação, a máquina virtual que executa o bytecode, os esquemas de assinatura, os principais mecanismos de segurança do sistema e o Smali, usado para ler e modificar o bytecode durante engenharia reversa.
