# Anexar o Objection ao App

Com o app já em execução, o Objection se anexa ao processo através do comando `explore`:

```bash
objection -g DVIA-v2 explore
```

Isso inicia o Objection e abre um console interativo para análise ao vivo do app, servindo como base para todos os comandos vistos nos próximos tópicos.
