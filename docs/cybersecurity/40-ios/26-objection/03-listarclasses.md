# Listar Classes Carregadas

Para inspecionar as classes Objective-C carregadas pelo app em execução:

```
ios hooking list classes
```

Isso retorna a lista completa de classes atualmente carregadas no runtime do app, servindo como ponto de partida para localizar a classe certa antes de explorar seus métodos ou aplicar um hook.
