# Monitorar Funções Criptográficas

Para monitorar as operações criptográficas realizadas pelo app:

```
ios monitor crypto
```

Esse comando permite observar e analisar funções criptográficas, como criptografia, descriptografia e hashing, conforme são executadas em tempo real. É particularmente útil para entender como o app lida com dados sensíveis, como senhas ou chaves, e para verificar se o algoritmo, o modo de operação ou o tamanho de chave usados são adequados.
