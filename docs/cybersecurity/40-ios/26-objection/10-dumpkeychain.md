# Dump do Keychain

Para extrair todos os dados armazenados no Keychain:

```
ios keychain dump
```

Isso retorna a lista completa de entradas do Keychain às quais o app tem acesso, revelando tokens, senhas e chaves guardadas ali, além das configurações de proteção aplicadas a cada item, o que ajuda a checar se estão de acordo com o que foi visto no capítulo de Secure Enclave.
