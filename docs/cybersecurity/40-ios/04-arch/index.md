# O que é o iOS?

O iOS (*iPhone Operating System*) é o sistema operacional para o dispositivo mobile da Apple, o iPhone. Além do iPhone, o iOS se tornou base para outros três sistemas operacionais desenvolvidos pela Apple como iPadOS, tvOS e watchOS.

# Arquitetura 
O iOS é estruturado em camadas que abstraem complexidade e organizam responsabilidades do sistema. Cada camada fornece serviços para a camada superior, permitindo modularidade, segurança e manutenção mais simples. As principais camadas são:
- Core OS (base do sistema)
- Core Services (serviços fundamentais)
- Media (renderização e multimídia)
- Cocoa Touch (interface e interação com o usuário)
