# Escrevendo Scripts para iOS

O Frida usa JavaScript para injetar código e hookar métodos de um app. Os dois exemplos a seguir mostram como escrever e rodar scripts básicos para iOS, um hookando um método em Objective-C e outro hookando um método em Swift.

- [Hookando um Método em Objective-C](/docs/cybersecurity/iossecurity/analisedinamica/frida/hookobjc)
- [Hookando um Método em Swift](/docs/cybersecurity/iossecurity/analisedinamica/frida/hookswift)
