# Bypass de Touch ID ou Face ID

Se o app usa Touch ID ou Face ID para autenticação, o Objection consegue contornar essa checagem:

```
ios ui biometric_bypass
```

Esse comando simula uma autenticação biométrica bem sucedida, o que é útil para avançar em fluxos protegidos por biometria sem que o dispositivo em uso realmente valide a impressão digital ou o rosto do usuário.
